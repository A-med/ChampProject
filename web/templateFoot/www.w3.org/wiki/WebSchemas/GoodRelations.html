This is an archived WebSchemas proposal GoodRelations for schema.org. See Proposals listing for more. Note: active schema.org development is now based at github




Overview

The schema.org team and Martin Hepp (Good Relations project lead) have been working on the integration the GoodRelations e-commerce vocabulary into schema.org.

The idea follows the precedent with IPTC's rNews, whose vocabulary was added in 2011. The basic idea is to make the comprehensive set of e-commerce elements from GoodRelations available directly from the schema.org namespace, so that Web developers can use GoodRelations in the very same way as the original e-commerce elements from schema.org.

GoodRelations will including mapping relations between both namespaces so that consuming clients can use the very same queries for processing schema.or data and GoodRelations data.
