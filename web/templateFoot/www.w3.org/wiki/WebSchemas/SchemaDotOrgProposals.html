Proposals for Schema.org

  • Since 2014 schema.org uses Github for active development (like many other W3C communities).
  • Email discussion remains in the WebSchemas public-vocabs list here at W3C.
  • Archived proposals 2011-2015 are archived here.
  • We now have a dedicated W3C Schema.org Community Group including a public-schemaorg@w3.org mailing list

For lots more detail please visit https://github.com/schemaorg/schemaorg/issues


From 2011-2014 this page collected public proposals for schema.org vocabulary extensions and improvements.

Until 2015, schema.org public discussions happened via the public-vocabs@w3.org W3C mailing list for discussions.
